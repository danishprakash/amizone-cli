# amizone-cli
A command line interface for interacting with [amizone.net](https://amizone.net/amizone/)

## Installation
```bash
$ git clone https://github.com/prakashdanish/amizone-cli.git

$ python3 acli.py
```

## Features
- My Courses
- Attendance
- My Faculty
- My Mentor

## Dependencies
- Selenium (webdriver, chrome or firefox webdriver)
